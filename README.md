# MyScore - 成绩管理系统

![Version](https://img.shields.io/badge/version-5.1-green)
![License](https://img.shields.io/badge/license-MIT-blue)

一个功能完善的考试成绩管理系统，支持雅思、大学英语四六级以及自定义考试的成绩录入、统计分析和数据管理。

## ✨ 功能特性

### 📊 支持的考试类型

#### 雅思考试 (IELTS)
- **听力 (Listening)**: 根据正确题数自动折算分数（0-40题，对应0.0-9.0分）
- **阅读 (Reading)**: 学术类阅读评分（0-40题，对应0.0-9.0分）
- **写作 (Writing)**: Task 1/Task 2 分离输入，自动加权计算总分
- **口语 (Speaking)**: 直接输入分数（0-9分，步长0.5）
- **Overall总分**: 自动计算四项平均分，按雅思规则取整

#### 大学英语考试 (CET)
- **四级 (CET-4)**: 完整的四级成绩管理
- **六级 (CET-6)**: 完整的六级成绩管理
- **听力**: 分三个部分计分
- **阅读**: 分三个部分计分
- **写作和翻译**: 支持原始分转换为报道分

#### 自定义考试
- **创建任意考试类型**: 自定义科目、计分规则
- **三种计分方式**: 直接输入 / 分部分计分 / 公式计算
- **完全可配置**: 科目名称、颜色、计算逻辑

### 📈 数据可视化
- **趋势图表**: 使用Chart.js绘制成绩趋势图
- **统计卡片**: 显示最佳成绩、平均成绩、考试次数
- **最近成绩**: 快速查看最新考试成绩
- **历史记录**: 完整的成绩历史列表

### 💾 数据管理
- **本地存储**: 使用LocalStorage浏览器本地存储
- **导出备份**: 支持将所有成绩导出为JSON文件
- **导入恢复**: 支持从JSON文件导入成绩，智能去重
- **数据持久化**: 关闭浏览器后数据不丢失

## 🚀 快速开始

### 1. 打开应用
直接在浏览器中打开 `myscore_V5.1.html` 文件即可使用。

### 2. 录入成绩
1. 点击顶部导航栏的"录入成绩"
2. 选择考试类型（雅思/四级/六级/自定义）
3. 填写考试日期
4. 依次填写各项成绩（手风琴自动展开收起）
5. 查看总分预览
6. 点击"保存成绩"

### 3. 查看统计
- 点击"仪表盘"查看统计信息
- 切换不同Tab查看各考试类型的详细数据

### 4. 数据备份与恢复
- **导出**: 在数据管理区域点击"导出备份"
- **导入**: 选择JSON文件导入，自动去重
