# MyScore - 成绩管理系统

![Version](https://img.shields.io/badge/version-5.3-green)
![License](https://img.shields.io/badge/license-MIT-blue)

一个功能完善的考试成绩管理系统，支持雅思、大学英语四六级以及自定义考试的成绩录入、统计分析和数据管理。

## ✨ 功能特性

### 📊 支持的考试类型

#### 雅思考试 (IELTS)
- **听力 (Listening)**: 根据正确题数自动折算分数（0-40题，对应0.0-9.0分）
- **阅读 (Reading)**: 学术类阅读评分（0-40题，对应0.0-9.0分）
- **写作 (Writing)**: Task 1/Task 2 分离输入，自动加权计算总分
- **口语 (Speaking)**: 直接输入分数（0-9分，步长0.5）
- **Overall总分**: 自动计算四项平均分，按雅思规则取整

#### 大学英语考试 (CET)
- **四级 (CET-4)**: 完整的四级成绩管理
- **六级 (CET-6)**: 完整的六级成绩管理
- **听力**: 分三个部分计分
- **阅读**: 分三个部分计分
- **写作和翻译**: 支持原始分转换为报道分

#### 自定义考试
- **创建任意考试类型**: 自定义科目、计分规则
- **四种计分方式**: 直接输入 / 多小题计分 / 分部分计分 / 公式计算
- **完全可配置**: 科目名称、颜色、计算逻辑

### 📈 数据可视化
- **趋势图表**: 使用Chart.js绘制成绩趋势图
- **统计卡片**: 显示最佳成绩、平均成绩、考试次数
- **最近成绩**: 快速查看最新考试成绩
- **历史记录**: 完整的成绩历史列表

### 💾 数据管理
- **本地存储**: 使用LocalStorage浏览器本地存储
- **导出备份**: 支持将所有成绩导出为JSON文件
- **导入恢复**: 支持从JSON文件导入成绩，智能去重
- **数据持久化**: 关闭浏览器后数据不丢失

## 🚀 快速开始

### 1. 打开应用
直接在浏览器中打开 `myscore_V5.3.html` 文件即可使用。

### 2. 录入成绩
1. 点击顶部导航栏的"录入成绩"
2. 选择考试类型（雅思/四级/六级/自定义）
3. 填写考试日期
4. 依次填写各项成绩（手风琴自动展开收起）
5. 查看总分预览
6. 点击"保存成绩"

### 3. 查看统计
- 点击"仪表盘"查看统计信息
- 切换不同Tab查看各考试类型的详细数据

### 4. 数据备份与恢复
- **导出**: 在数据管理区域点击"导出备份"
- **导入**: 选择JSON文件导入，自动去重

## 📖 详细功能说明

### 自定义考试类型使用指南

自定义考试允许你创建任意类型的考试评分系统，比如托福、GRE、模拟测试等。

#### 1. 创建新考试类型

1. 点击顶部"自定义考试"导航
2. 点击"+ 新建考试"按钮
3. 填写考试名称（如：托福）和描述（可选）

#### 2. 添加科目

每个考试可以添加多个科目，每个科目需要设置：

| 设置项 | 说明 |
|-------|------|
| 科目名称 | 如：阅读、听力、写作 |
| 简称 | 显示在卡片上的缩写，如：阅、听、写 |
| 颜色 | 用于区分不同科目的标识色 |
| 计分方式 | 四种方式可选（见下文） |

#### 3. 计分方式详解

**方式一：直接输入分数**
- 适用场景：直接输入最终成绩
- 示例：雅思口语直接输入 7.5 分
- 配置：只需设置最小值、最大值（如 0-9）

**方式二：多小题计分（V5.2新增）**
- 适用场景：一个大题包含多个小题，需要分别计分后加总
- 示例：数学解答题20分 = 第一问5分 + 第二问7分 + 第三问8分
- 配置：
  - 添加多个"小题"
  - 每小题设置：名称（如：第一问）、满分值（如：5）
  - 系统自动计算：各小题得分之和 = 该大题总分

**方式三：分部分计分**
- 适用场景：试卷分多个部分，每部分题数和分值不同
- 示例：四六级听力（短对话8题×7.1分 + 长对话7题×7.1分 + 短文10题×14.2分）
- 配置：
  - 添加多个"部分"
  - 每部分设置：名称（如：短对话）、每题分值（如：7.1）、最大题数（如：8）
  - 系统自动计算：答对题数 × 每题分值，各部分的和即为该科目总分

**方式四：公式计算**
- 适用场景：原始分需要乘以系数换算
- 示例：四六级写作原始分0-15分，报道分 = 原始分 × 7.1
- 配置：
  - 原始分最大值（如：15）
  - 乘数系数（如：7.1）
  - 系统自动计算：输入原始分 × 乘数 = 最终分数

#### 4. 计算总分

创建考试时默认开启"计算总分"，系统会将各科分数相加得到考试总分。如需关闭，目前版本暂不支持，可通过创建单科目考试来实现。

#### 5. 使用自定义考试

创建完成后：
1. 进入"录入成绩"页面
2. 考试类型选择区域会显示你创建的考试
3. 点击选择后，表单会按照你设置的计分方式呈现
4. 录入成绩并保存

#### 6. 删除自定义考试

在"自定义考试"页面点击考试卡片右上角的"×"即可删除。**注意**：删除后该考试的历史成绩也会被一并删除。
