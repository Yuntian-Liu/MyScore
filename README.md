# MyScore - 成绩管理系统

![Version](https://img.shields.io/badge/version-4.0-green)
![License](https://img.shields.io/badge/license-MIT-blue)

一个功能完善的考试成绩管理系统，支持雅思和大学英语四六级的成绩录入、统计分析和数据管理。

## ✨ 功能特性

### 📊 支持的考试类型

#### 雅思考试 (IELTS)
- **听力 (Listening)**: 根据正确题数自动折算分数（0-40题，对应0.0-9.0分）
- **阅读 (Reading)**: 学术类阅读评分（0-40题，对应0.0-9.0分）
- **写作 (Writing)**: 支持直接输入总分或通过Task 1/Task 2自动计算
- **口语 (Speaking)**: 直接输入分数（0-9分，步长0.5）
- **Overall总分**: 自动计算四项平均分，按雅思规则取整

#### 大学英语考试 (CET)
- **四级 (CET-4)**: 完整的四级成绩管理
- **六级 (CET-6)**: 完整的六级成绩管理
- **听力**: 分三个部分计分（短对话/新闻、长对话、短文/讲座）
- **阅读**: 分三个部分计分（选词填空、长篇阅读、仔细阅读）
- **写作和翻译**: 支持原始分转换为报道分

### 📈 数据可视化
- **趋势图表**: 使用Chart.js绘制成绩趋势图
- **统计卡片**: 显示最佳成绩、平均成绩、考试次数
- **最近成绩**: 快速查看最新考试成绩
- **历史记录**: 完整的成绩历史列表

### 💾 数据管理
- **本地存储**: 使用LocalStorage浏览器本地存储，数据安全可靠
- **导出备份**: 支持将所有成绩导出为JSON文件
- **导入恢复**: 支持从JSON文件导入成绩，智能去重
- **数据持久化**: 关闭浏览器后数据不丢失

## 🚀 快速开始

### 1. 打开应用
直接在浏览器中打开 `myscore_V4.0.html` 文件即可使用。

### 2. 录入成绩

#### 雅思成绩录入步骤：
1. 点击顶部导航栏的"录入成绩"按钮
2. 选择"雅思"考试类型
3. 填写考试日期
4. 依次填写各项成绩：
   - **听力**: 输入正确题数（0-40），系统自动计算分数
   - **阅读**: 输入正确题数（0-40），系统自动计算分数
   - **写作**: 方式一：直接输入总分（0-9分，步长0.5）
              方式二：输入Task 1和Task 2分数，系统自动计算总分
   - **口语**: 输入口语分数（0-9分，步长0.5）
5. 查看Overall总分预览
6. 点击"保存成绩"按钮

#### 四六级成绩录入步骤：
1. 点击顶部导航栏的"录入成绩"按钮
2. 选择"四级(CET-4)"或"六级(CET-6)"
3. 填写考试日期
4. 依次填写各项成绩：
   - **听力**:
     - 四级：短对话/新闻（0-8题）、长对话（0-7题）、短文（0-10题）
     - 六级：长对话（0-8题）、短文（0-7题）、讲座/报道（0-10题）
   - **阅读**: 选词填空（0-10题）、长篇阅读（0-10题）、仔细阅读（0-10题）
   - **写作**: 输入原始分（0-15分），系统自动转换为报道分
   - **翻译**: 输入原始分（0-15分），系统自动转换为报道分
5. 查看总分预览
6. 点击"保存成绩"按钮

### 3. 查看统计分析
- 点击"仪表盘"按钮查看统计信息
- 切换不同的Tab查看：
  - **概览**: 最近成绩和总考试次数
  - **雅思统计**: 雅思最佳/平均成绩、趋势图
  - **四六级统计**: 四级和六级的详细统计数据和趋势图

### 4. 数据备份与恢复

#### 导出数据：
1. 在仪表盘页面找到"数据管理"区域
2. 点击"导出备份"按钮
3. 系统会自动下载JSON格式备份文件

#### 导入数据：
1. 在仪表盘页面找到"数据管理"区域
2. 点击"选择 JSON 文件导入"
3. 选择之前导出的JSON文件
4. 系统自动导入并去重，显示导入结果

## 📋 使用规范

### 输入规范
- 所有分数输入必须符合题目要求的范围
- 雅思写作和口语分数支持0.5步长
- 考试日期格式：YYYY-MM-DD
- 系统会实时验证输入并给出提示

### 数据安全
- 建议定期导出数据备份
- 导出的JSON文件保存在安全位置
- 更换浏览器或电脑时，可以通过导入功能恢复数据

### 分数计算规则
- **雅思听力/阅读**: 严格按照官方评分表计算
- **雅思Overall**: 四项平均后按规则取整（0.25进0.5，0.75进1.0）
- **雅思写作**: Task 1权重1/3，Task 2权重2/3
- **四六级**: 各部分按官方分值比例转换为报道分

## 🛠️ 技术栈

- **HTML5**: 页面结构
- **Tailwind CSS**: 样式框架（通过CDN引入）
- **Chart.js**: 数据可视化库（通过CDN引入）
- **Vanilla JavaScript**: 纯JavaScript实现所有功能
- **LocalStorage**: 浏览器本地存储

## 💡 使用建议

1. **定期备份**: 每次考试后及时导出数据备份
2. **准确录入**: 确保输入的题数和分数准确无误
3. **趋势分析**: 通过趋势图了解自己的成绩变化
4. **目标设定**: 根据统计数据制定合理的学习目标

## 📝 注意事项

- 本系统使用LocalStorage存储数据，清空浏览器缓存会导致数据丢失
- 建议使用现代浏览器（Chrome、Firefox、Edge、Safari）获得最佳体验
- 导入数据时会自动过滤重复记录（基于记录ID）
- 清空所有记录的操作不可恢复，请谨慎使用
